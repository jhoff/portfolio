Crappy Keyboard

An introduction into HTML5

Crappy Keyboard is a small test page to see what I could do with some HTML5 elements.

I mostly focused on the canvas element and it's basic drawing capabilities.

It pulls random tweets from celebrities and makes you retype them. Meanwhile, they "keys" get worn out and stop working after a while. It really wasn't originally intended to be a game, but its good for a chuckle.

What it is:

  • Lots of Dynamic HTML5 elements
  • Unobtrusive library-less Javascript ( with the exception of 1 jQuery ajax call )
  • Written in less than 10 hours

What it's NOT:

  • Efficient - Goal was to make it work
  • Cross Browser Compatible - It's only tested in Firefox 3.6
  • Particularly Useful

Takeaways:

  • Displays basic understanding of Javascript
  • Javascript builds the vast majority of the page at load.
  • Coding style ( note the lack of focus on efficiency )
